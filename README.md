# 2D 物理沙盒 🏖️

开发工具💻：C/C++ SDL3 CMake

编译器📝：Emacs

## 概述📋

一个简单模拟**重力掉落**和**碰撞计算**的2D物理沙盒。

## 操作指南🕹️

调整方块（沙子）大小

>`q` 减少宽度，`w` 增加宽度
>
>`e` 减少高度，`r` 增加高度

调整方块（沙子）质量

> `a` 减少质量，`s` 增加质量

调整方块（沙子）恢复系数

> `d` 减少系数，`f` 增加系数

调整方块（沙子）摩擦系数

> `z` 减少系数，`x` 增加系数

调整方块（沙子）颜色

>`y` 减少红色值，`u` 增加红色值
>
>`h` 减少绿色值，`j` 增加绿色值
>
>`n` 减少蓝色值，`m` 增加蓝色值

调整方块（沙子）重力大小

>`v` 减少重力，`b` 增加重力

调整fps

> `[` 增加fps值，`]` 减少fps值
